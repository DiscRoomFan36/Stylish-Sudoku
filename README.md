
# Stylish Sudoku

A simple stylish sudoku player.

![Stylish Sudoku Player Still Image](./readme_assets/nice_display_image.png)


## Quick Start

```bash
# compile the build script once.
$ cc -o nob nob.c

# compile and run
$ ./nob debug && ./build/main_debug
```
